Inline vs Block

  • Block Elements:

    Takes up the full-width available(whole Block)
    start from new

    • h1 to h6 Tag
    • div tag
    • section tag
    • Etc.

  • Inline Elements:

    Takes up only necessary width
    don't start from new line

    • Anchor Tag
    • span Tag
    • image Tag
    • Etc.

Html Entities

An HTML Entities is a piece of text that begins with an ampersand (&) and ends with a semicolon(;)

Used to display reserved charcters and invisible character

  • for < use = & lt (less than)
  • for > use = & gt (grater than)
